## Prislista

Nedan hittar du våra vanligaste priser. Alla priser är inklusive moms. Tandvården omfattas av det statliga tandvårdsstödet och högkostnadsskyddet.

### Undersökning och diagnostik

| Behandling                   | Pris    |
| ---------------------------- | ------- |
| Basundersökning (ny patient) | 985 kr  |
| Revisionsundersökning        | 785 kr  |
| Akut undersökning            | 5600 kr |
| Röntgen per bild             | 75 kr   |
| Panoramaröntgen              | 620 kr  |

### Förebyggande behandling

| Behandling                       | Pris     |
| -------------------------------- | -------- |
| Tandhygienistbehandling (60 min) | 1 250 kr |
| Fissurförsegling per tand        | 390 kr   |
| Fluorbehandling                  | 215 kr   |

### Lagningar och restaurationer

| Behandling              | Pris     |
| ----------------------- | -------- |
| Kompositfyllning, liten | 825 kr   |
| Kompositfyllning, stor  | 1 450 kr |
| Krona (helkeramisk)     | 5 950 kr |
| Fasad                   | 5 950 kr |

### Tandvårdsstöd

Som patient hos oss har du rätt till det statliga tandvårdsstödet. Det innebär att du får ett allmänt tandvårdsbidrag (ATB) på 300 eller 600 kr per år beroende på ålder. Dessutom finns högkostnadsskyddet som ger dig rabatt om dina tandvårdskostnader överstiger 3 000 kr under en tolvmånadersperiod.

Kontakta oss gärna om du har frågor om priser eller finansiering.
